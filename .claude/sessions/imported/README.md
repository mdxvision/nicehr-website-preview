# Imported Conversations

This folder is for conversations imported from Claude.ai or other sources.

## How to Import

1. Go to claude.ai and open the conversation
2. Copy the relevant content
3. Create a new .md file here with a descriptive name
4. Paste and format as markdown

## Naming Convention

```
YYYY-MM-DD-topic-description.md
```

Example: `2025-12-15-initial-website-planning.md`

## Referencing in CLAUDE.md

Add imports to the main CLAUDE.md:

```markdown
See @.claude/sessions/imported/2025-12-15-initial-website-planning.md
```
